# electricvehicle
precise and quick car.
Change distance variable in code depending on target distance.

## Parts used:
4x 2 7/8 in diameter banebots wheels

2x 4mm axles

4x 8mm od/4mm id bearings

1x 600ppr rotary encoder

1x 12V DC Motor

lever connectors

jumper wires

button 

8AA Batteries + battery pack

arduino uno

L298N
